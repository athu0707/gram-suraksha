# 🏛️ Gram Suraksha App
### ग्राम सुरक्षा – Village Issue Reporting System

A full-stack web application for villagers to report local issues and for authorities to resolve them.

---

## 📁 Project Structure

```
gram_suraksha/
├── backend/
│   ├── main.py              # FastAPI app entry point
│   ├── database.py          # SQLite database setup
│   ├── requirements.txt     # Python dependencies
│   ├── models/
│   │   └── __init__.py      # Database models (User, Complaint)
│   ├── schemas/
│   │   └── __init__.py      # Pydantic request/response schemas
│   ├── routers/
│   │   ├── auth.py          # Login & registration endpoints
│   │   ├── complaints.py    # Complaint CRUD endpoints
│   │   ├── admin.py         # Admin panel endpoints
│   │   └── users.py         # Profile, leaderboard, notifications

│   │   ├── auth.py          # JWT auth + password hashing
│   │   └── rewards.py       # Points, badges, penalty system
│   └── uploads/             # Uploaded images stored here
└── frontend/
    └── templates/
        └── index.html       # Complete mobile-friendly UI
```

## ⚙️ Setup Instructions

### Step 1 – Prerequisites
Make sure you have **Python 3.9+** installed:
```bash
python --version
```

### Step 2 – Create Virtual Environment
```bash
cd gram_suraksha/backend
python -m venv venv

# Activate on Windows:
venv\Scripts\activate

# Activate on Mac/Linux:
source venv/bin/activate
```

### Step 3 – Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 4 – Run the Server
```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### Step 5 – Open the App
Open your browser and go to:
```
http://localhost:8000
```

## 🔑 First Time Setup

### Create an Admin Account
1. Open the app → Click **Admin** tab
2. Click "Register Admin Account"
3. Fill in admin details and login

### Register as a Villager
1. Click **Register** tab
2. Fill in your details
3. Start reporting issues!

## 🌐 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/auth/register` | Register new user |
| POST | `/api/auth/login` | Login |
| POST | `/api/complaints/` | Submit complaint |
| GET | `/api/complaints/my` | Get my complaints |
| GET | `/api/admin/complaints` | Get all (admin only) |
| PATCH | `/api/admin/complaints/{id}` | Update status (admin only) |
| GET | `/api/users/leaderboard` | Top users leaderboard |
| GET | `/api/users/notifications` | My notifications |

### Interactive API Docs (auto-generated by FastAPI):
```
http://localhost:8000/docs
```

## 🏆 Reward System

| Action | Points |
|--------|--------|
| Complaint marked Valid | +10 |
| Complaint Resolved | +20 |
| Invalid/Fake complaint | -10 |

### Badge Levels
| Badge | Points Required |
|-------|----------------|
| 🌱 Beginner | 0 |
| 🙋 Active Citizen | 30 |
| ⭐ Top Reporter | 100 |
| 🛡️ Village Guardian | 200 |

### Penalty System
- 3 invalid complaints → Account blocked

## 🛠️ Tech Stack
- **Backend**: FastAPI (Python)
- **Database**: SQLite (via SQLAlchemy ORM)
- **Auth**: JWT tokens + bcrypt password hashing
- **Frontend**: HTML5 + CSS3 + Vanilla JavaScript
- **Image Upload**: multipart/form-data
